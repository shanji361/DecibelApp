# Decibel Detector App
## Features

- Real-time sound level detection using the device microphone.

- Displays decibel levels numerically and with a colored progress bar.

- Alerts when noise exceeds a threshold (75 dB ).

- Handles microphone permissions.

- Features Start/Stop recording button.

- Logs RMS, dB, and normalized decibel values for debugging.

---

## How to Run
1. Clone this repository:
   ```
   git clone https://github.com/shanji361/DecibelApp.git
   ```
2. Open the project in Android Studio.

3. Run the app on an emulator or a physical Android device.
